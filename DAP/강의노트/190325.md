# 행렬
- 2차원 벡터로 정의 됨
## 벡터를 행렬로 바꾸는 법
- dim(v) : 해당 배열의 차원 반환
- dim(v) <- c(m,n) : m*n의 행렬로 바꿈
- 두 개의 벡터를 결합한 행렬 : rbind(vector, vector) / cbind(vector, vector) -
  두 벡터를 열의 방향으로 결합, 벡터가 저장된 변수의 이름이 행/열의 이름으로 사용
- 벡터와 행렬, 행렬과 행렬의 결합도 가능 : 마찬가지로 rbind / cbind 이용
## matrix 함수를 사용하는 법
- matrix(data, nrow, ncol, byrow, dimnames)
- 열의 방향으로 우선적으로 데이터 채움 -> byrow = TRUE 옵션으로 바꿀 수 있음
- [n,]의 경우에는 행, [,n]의 경우에는 열임을 뜻함
- dimnames=list(행이름벡터,열이름벡터)로 행열의 이름을 각각 설정 가능
- data인수에 하나의 값만 주면 재사용규칙으로 인해 나머지 값들도 동일한 값으로 채워짐 -> 0, NA등으로 초기화 가능
- nrow나 ncol중 하나의 인수만 입력해서 줄 경우 자동으로 숫자를 맞춰서 행/열을 생성한다
- str() 함수를 이용하면 내부구조를 볼 수 있음
- ncol() , nrow(), length() 등의 함수로 행/열/전체 길이 확인 가능
## 행렬 인덱싱
- 차원에 대한 구분은 ,로, mat[m,n]의 식으로 사용 ... 생략가능(전체 행/열 선택 기능)
- 행이나 열을 추출 할 시 벡터로 반환 -> 2차원으로 유지하게 하려면 drop=FALSE 옵션 사용
- row나 col 인자에 m:n 식으로 여러개의 행또는 열 선택 가능
- 음수로 옵션을 줄 시, 해당 행/열을 선택하지 않는다는것을 뜻함
- 이 인덱스를 활용해 값 변경도 가능, 열 전체 행 전체를 바꾸는 것도 가능
- colnames(), rownames()를 이용해 행 이름 열 이름 지정 후, 이를 인덱스로 사용 할 수 도 있음
## 연산
- 행렬과 단일숫자의 연산 : 각 요소별로 단일숫자와 연산 한 값을 저장
- 행렬과 행렬 간의 연산 : 차원이 일치해야 가능(일대일 대응 하는 원소들끼리 연산한 결과 저장)
- 일반적인 행렬곱셈 : % * % 연산자 사용
- 행렬과 벡터 간의 연산 : 벡터를 해당 행렬의 차원으로 바꿔서 연산 -> 원소의 개수가 부족하면 재사용법칙 적용
- 평균값, 총합 구하기 : rowMeans(), colMeans() / rowSums(), colSums()
- t() 함수를 이용하면 전치행렬 생성 ... 하나의 행을 추출해서 이를 전치행렬로 변환할시 열으로 구성이 안됨(t 함수가 벡터를 하나의 열로 이루어진 행렬로 간주)
