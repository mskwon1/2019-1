# Vector
- 한가지 유형의 데이터만 담을 수 있음
- c() 함수로 생성 : concatenate 라는 뜻
- Colon 사용시 해당 range의 숫자들이 모두 들어감
- seq(from=int, to=int, length.out=int, by=int) : from 부터 to까지 같은간격의 length.out개 숫자들 / by 간격의 숫자들
- rep(x, times=int, each=int, length.out=int) : x를 times만큼 반복 / 요소별로 each만큼 반복 / length.out 개만큼 반복
  times에 벡터를 넣을 수 있음 : 해당되는 인덱스의 숫자만큼 해당 인덱스의 요소 반복
## 벡터 연산
- 사칙연산 지원(+,-,x,/,%/%몫,%%나머지)
- 벡터 사이의 연산은 원소대 원소로 연산 후 벡터로 묶음 (각각 연산), 수행하는 벡터들의 원소의 수가 서로 배수/약수여야함
  -> 아닐 시 Recycling Rule에 따라 불완전한 연산 수행(순서대로 다시 처음으로 돌아가서 반복 사용 / 연산 수행)
- 할당연산자 <- 와 = 은 같음
- 논리연산 지원 (==, !=, <, >, |, &) TRUE는 1, FALSE는 0으로 생각
  -> TRUE인 조건의 개수를 + 연산으로 구할 수 있음, 평균계산으로  TURE 비율도 계산 가능
- sqrt() 제곱근 연산 : R 내부에서 숫자들은 부동소수점 수로 계산됨
-> 오차가 존재함 -> all.equal(...,...) 함수로 아주 작은 반올림은 무시
- paste(...,...) : 문자열 결합
